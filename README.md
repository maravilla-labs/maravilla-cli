# maravilla-cli

Public CLI assets and AI agent skills for [Maravilla Cloud](https://www.maravilla.cloud).

This repository hosts:

- **[`packages/maravilla-skills`](./packages/maravilla-skills)** — AI agent skills for building Maravilla Cloud apps. Works with Claude Code, Cursor, GitHub Copilot, and 30+ other AI coding tools.

## Activate skills in your project

```bash
npx skills add maravilla-labs/maravilla-cli/packages/maravilla-skills
```

This installs all 14 Maravilla Cloud skills into your project's agent configuration so your assistant has accurate, source-of-truth knowledge about the SDK, auth contract, policies, events, workflows, realtime, push, and framework adapters.
